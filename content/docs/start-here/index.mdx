如果你刚开始系统学习 AI，请先完成一个具体任务，再逐步扩展工具和工作流。

<Cards>
  <Card title="用 20 分钟完成第一个 AI 任务" href="/docs/start-here/first-ai-task/">
    用提问、校验和改写完成一次低风险实践。
  </Card>
</Cards>

1. 选择一个高频、低风险的任务。
2. 阅读对应工具教程，建立可重复的操作步骤。
3. 使用场景打法完成一次真实实践。
4. 把有效提示词和结论整理为自己的参考资料。

下一步可从[工具教程](/docs/tools/)或[场景打法](/docs/playbooks/)开始。
