## 全屏与性能

图表接近视口时再加载，展开后会重新测量容器；全屏 SVG 不继承工具按钮的图标尺寸。关闭时销毁监听器，并把焦点还给展开按钮。

<Callout type="success" title="最低完整性">颜色之外还要有标签或数值；无 JavaScript 时仍显示标题、说明和数据表；减少动态时停止非必要过渡。</Callout>
