适合发布前验收。目标不是让每个状态看起来完全相同，而是让内容、操作和位置感在不同能力条件下保持等价。

<KeyTakeaways title="四条不可退让的基线">
  <li>页面不能产生视口级横向滚动。</li>
  <li>每个操作都能用键盘完成，并有可见焦点。</li>
  <li>图片、图表、视频和交互都有文字语义或静态回退。</li>
  <li>减少动态、无 JavaScript 和打印状态仍能获得关键内容。</li>
</KeyTakeaways>
