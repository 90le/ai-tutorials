## 覆盖范围

| 能力层 | 展示内容 | 验收重点 |
| --- | --- | --- |
| 文章语言 | 标题、列表、引用、表格、代码、脚注 | 连续阅读与移动端边界 |
| 结构表达 | 提示、步骤、页签、折叠、类型表 | 语义与键盘路径 |
| 图示数据 | Mermaid、ECharts、公式、数据表、下载 | 来源、降级与全屏查看 |
| 图片证据 | 单图、组图、前后对比、灯箱 | 替代文本、缩放与焦点恢复 |
| 视频 | 本地 MP4、海报、字幕、文字稿 | 不自动播放与等价替代内容 |
| 页面系统 | 知识树、本文目录、进度、主题、打印 | 无溢出、无 404、无控制台错误 |

### 怎么使用这座实验室

读者可以把它当作阅读能力说明；作者可以在写 MDX 前确认组件边界；维护者则应在主题或组件变更后，用它完成视觉回归，而不是另外维护一套脱离真实文章的演示应用。

所有示意数据都会明确标注，不把视觉完整度伪装成事实可信度。
