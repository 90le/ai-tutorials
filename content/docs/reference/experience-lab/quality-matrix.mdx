## 自适应阅读版心

正文使用三条内容轨道：连续文字走 `prose`，图表和代码走 `wide`，明确标记的沉浸媒体可以走 `full`。左侧知识树或右侧本文目录收起后，对应宽度会归还给文章，而不是留下一块空白。

| 左侧知识树 | 右侧目录 | 文章最大宽度 |
| --- | --- | --- |
| 展开 | 展开 | 60rem |
| 任一收起 | 另一侧展开 | 68rem |

## 代码工具栏

```ts title="evidence-pipeline.ts" {3-7}
type Evidence = { claim: string; source: string; verified: boolean };

export function acceptEvidence(item: Evidence) {
  if (!item.source) throw new Error('缺少来源');
  if (!item.verified) return { state: 'needs-review', item };
  return { state: 'accepted', item };
}
```

工具栏由 React 直接渲染，和复制按钮位于同一标题行。它支持自动换行、显示行号、展开全部；窄屏时工具栏自身横向滚动，不覆盖标题或代码。

## 无障碍与降级

<ul className="docs-experience-checklist">
  <li><strong>键盘：</strong>跳转链接、按钮、滑块、折叠、画布和灯箱都有可见焦点。</li>
  <li><strong>语义：</strong>图表有名称，图片有替代文本，表格有标题和表头。</li>
  <li><strong>性能：</strong>ECharts 与 Three.js 接近视口后动态加载。</li>
  <li><strong>减少动态：</strong>不依赖动画表达唯一信息。</li>
  <li><strong>打印：</strong>隐藏浮层工具，保留正文、图注、表格与来源。</li>
  <li><strong>静态导出：</strong>不依赖 Cookie、登录、私有 API 或服务器运行时。</li>
</ul>
