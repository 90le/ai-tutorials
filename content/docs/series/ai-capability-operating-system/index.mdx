这是一条面向真实产出的学习路线，而不是产品功能目录。建议每个模块都选择一个自己的真实任务完成练习。

## 学习路线

1. [第一次把 AI 当作工作系统](/docs/start-here/ai-task-operating-system/)：建立任务简报、证据边界和验收。
2. [AI 深度研究完整工作流](/docs/tools/chatgpt/deep-research-workflow/)：把互联网信息整理成可复核证据。
3. [搭建 AI 第二大脑](/docs/playbooks/ai-second-brain/)：让资料、笔记与输出形成长期循环。
4. [智能体编程的仓库级交付](/docs/tools/codex/agentic-repository-delivery/)：在代码仓库中管理 Agent 的实现与验证。
5. [为 AI Agent 设计工具](/docs/reference/agent-ready-toolkit/)：把稳定经验封装成规则、技能与接口。
6. [AI 视觉创作工作流](/docs/playbooks/ai-visual-production/)：完成带来源记录和发布质检的视觉资产。

## 最终练习

选择一个公开主题，完成一篇图文教程：先写任务合同，进行多来源研究，把资料进入知识库，用 Agent 协助维护仓库，制作原创视觉并完成三项构建验证。最终交付必须能反向追溯到来源、规则和验证记录。
