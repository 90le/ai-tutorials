# 生成教程视觉素材

为指定教程生成或编辑一张适合公开发布的位图视觉素材。

1. 先读取 `AGENTS.md`，确认素材用于封面、章节插图、概念图还是教程示意图；只有位图确有价值时才使用 ImageGen，流程图、简单图标和可访问的结构优先由 HTML/CSS、Mermaid 或已有图标实现。
2. 使用 ImageGen 的内置生成/编辑流程。提示词说明用途、画面主体、构图、色彩、文字是否精确出现，以及必须避免的元素；不要伪造商标、使用水印或把重要教程文字烘焙进图片。
3. 选择最终版本后，保存到 `public/images/<topic>/<descriptive-name>.png` 或 `.webp`，不覆盖已有素材；在文章中加入有意义的替代文本。
4. 检查图片在目标页面的裁切、响应式尺寸和对比度，随后运行 `npm run lint`、`npm run types:check` 和 `npm run build`。
5. 报告最终素材路径、使用的生成提示词和页面引用位置。
