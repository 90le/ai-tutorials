# 制作教程配套演示

把一篇已完成的长教程整理为配套的 Dashi HTML 演示；文档页仍是主内容，演示只服务于讲解、分享或培训。

1. 先读取 `AGENTS.md` 和目标教程，提炼听众、目标、核心结论、页数和需要展示的数据。没有明确授权时，询问用户演示风格以及是否需要图片/视频；用户明确“都你来定”时再自行选择。
2. 使用 Dashi PPT 的 JSON 计划与校验工作流生成演示。每页只承载一个主要信息，不复用模板默认文案，也不手写自由 HTML slide。
3. 将可发布的 HTML 演示及其素材放到 `public/decks/<deck-slug>/`；在对应教程中加入经过本地构建验证的链接。除非用户明确要求，不生成或提交 PPTX/PDF。
4. 如需原创位图插图，使用 ImageGen 生成并按 Dashi 的媒体暂存规则写入演示。
5. 运行 Dashi 要求的 deck 校验，以及 `npm run lint`、`npm run types:check` 和 `npm run build`。报告演示路径、预览方式、主题选择和验证结果。
